Before running any of the end solutions provided in this hands-on lab you need to install the NuGet package dependencies. After opening the solutions with Visual Studio, follow these steps:

1. In visual Studio, right-click the solution and select Enable NuGet Package Restore. Click Yes if you are asked for a confirmation.
2. Right-click the solution and select Manage NuGet Packages for Solution.
3. Click Restore to restore missing references.
4. Build the solution. The NuGet dependencies will be downloaded and installed automatically.

For more information see: http://docs.nuget.org/docs/workflows/using-nuget-without-committing-packages
